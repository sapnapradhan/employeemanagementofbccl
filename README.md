# 🏢 BCCL Employee Management System

A modern and user-friendly **Employee Management System** developed for **BCCL (Bharat Coking Coal Limited)**. This application simplifies employee record management by providing an efficient interface to add, update, view, and manage employee information.

---

## 🚀 Live Demo

You can access the deployed website here:

**https://employeemanagementofbccl.vercel.app/**

---

## ✨ Features

* 👤 Employee Registration
* 📋 View Employee Details
* ✏️ Update Employee Information
* 🗑️ Delete Employee Records
* 🔍 Search and Filter Employees
* 📱 Responsive User Interface
* ⚡ Fast and Easy Navigation

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Language:** JavaScript
* **Build Tool:** Vite
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

---

## 📦 Installation

Clone the repository and run it locally:

```bash
git clone https://github.com/sapnapradhan/employeemanagementofbccl.git
cd employeemanagementofbccl
npm install
npm run dev
```
